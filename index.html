DevOps com GitHub Actions

Automatização de tarefas no desenvolvimento moderno

O que é isto?

Este site é atualizado automaticamente sempre que há mudanças no código. Isso é possível graças ao GitHub Actions.

O que acontece?

  • Fazemos push no código
  • O GitHub Actions executa um workflow
  • O site é publicado automaticamente
